# LBA Task Reassignment Module

## Overview
This module provides advanced task reassignment functionality with time data handling for the LBA system.

## Features
- Reassign tasks with multiple time handling options
- Bulk reassignment capabilities
- Full audit trail
- Performance optimized for large datasets
- Permission-based access control

## Permissions
| Role | Access Level |
|------|-------------|
| PMO | Full access to all tasks |
| Project Manager | Tasks in their projects |
| Department Manager | Tasks in their department |
| Team Lead | Tasks of their team members |

## Time Transfer Options
1. **Keep Time**: Keep time entries with original assignee
2. **Transfer Time**: Move all time entries to new assignee
3. **Split Time**: Split time entries between assignees

## Installation
1. Copy module to addons directory
2. Update module list
3. Install module

## Usage
1. Open any task
2. Click "Reassign" button
3. Select new assignee and time handling option
4. Preview changes and confirm
